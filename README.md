    - wallust
    - fastfetch

    - dunst
    - switch
    - swww
    - hyprctl
    - imagemagick
    - kitty
    - nwg-look
    - wofi
    - swaync
    - waytrogen

## **How to install wallpaper**
type `"swww img <file>"`
then press `Super + O` to sync theme
## **Binds**
`Super + O` - Terminal
`Super + C` - Kill active
`Super + F` - Fullscreen
`PrintScr` - Screenshot using `~/.config/scripts/utils/screenshot.sh`
and another (in `~/.config/hypr/keybinds/keybinds.conf`)
